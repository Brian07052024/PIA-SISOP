Comandos en Linux

¿Qué es un comando?

Un comando Linux es un programa o utilidad que se ejecuta en la línea de comandos. Una línea de comandos es una interfaz que acepta líneas de texto y las procesa en forma de instrucciones para tu ordenador.

Cualquier interfaz gráfica de usuario (GUI) no es más que una abstracción de los programas de línea de comandos. Por ejemplo, cuando cierras una ventana haciendo clic en la «X», hay un comando que se ejecuta detrás de esa acción.

Los comandos de Linux más utilizados

Antes de pasar a los comandos más utilizados de Linux, asegúrate de abrir un terminal. En la mayoría de las distribuciones de Linux, usarás Ctrl + Alt + T para hacerlo. Si esto no funciona, busca en tu panel de aplicaciones «terminal».

Navegación y manejo de archivos

A continuación se muestran algunos comandos y opciones para navegar y manejar archivos en la terminal de Linux.

pwd: Muestra el directorio actual en el que estás trabajando (ruta completa).

ls: Lista los archivos y carpetas del directorio actual. Con -l, muestra detalles adicionales como permisos, propietario, tamaño y fecha de modificación.

cd: Cambia de directorio. Por ejemplo, cd /home/usuario/Documentos te moverá al directorio "Documentos".

mkdir: Crea un nuevo directorio. Por ejemplo, mkdir proyecto crea una carpeta llamada "proyecto".

rm: Elimina archivos. Por ejemplo, rm archivo.txt borra el archivo "archivo.txt".

cp: Copia archivos de un lugar a otro. Por ejemplo, cp archivo.txt /home/usuario/Documentos/ copia el archivo "archivo.txt" al directorio "Documentos".

mv: Mueve o renombra archivos. Por ejemplo, mv archivo.txt nuevo_archivo.txt cambia el nombre de "archivo.txt" a "nuevo_archivo.txt".

Visualización y edición de archivos

A continuación se muestran algunos comandos y opciones para visualizar y editar archivos en la terminal de Linux.

cat: Muestra el contenido completo de un archivo directamente en la terminal. Por ejemplo, cat archivo.txt mostrará el texto del archivo "archivo.txt".

more: Muestra el contenido de un archivo página por página. Por ejemplo, more archivo.txt permite leer el archivo "archivo.txt" de manera que el contenido no se desborde en la pantalla.

less: Similar a more, pero permite desplazarse hacia arriba y hacia abajo en el archivo. Por ejemplo, less archivo.txt mostrará el contenido del archivo "archivo.txt" con más control para navegar.

nano: Abre el editor de texto nano, que permite visualizar y editar archivos en la terminal. Por ejemplo, nano archivo.txt abrirá el archivo "archivo.txt" en nano para su edición.

Gestión de permisos y procesos

A continuación se describen comandos básicos para manejar permisos de archivos y gestionar procesos en la terminal de Linux.

chmod: Cambia los permisos de un archivo. Por ejemplo, chmod 755 script.sh otorga permisos de lectura y ejecución a todos los usuarios, pero solo permisos de escritura al propietario del archivo.

chown: Cambia el propietario y grupo de un archivo. Por ejemplo, chown usuario:grupo script.sh asigna "usuario" como propietario y "grupo" como grupo del archivo "script.sh".

ps aux: Muestra una lista de todos los procesos que se están ejecutando en el sistema, junto con información como el usuario, el uso de memoria y CPU, y el comando que inició el proceso.

kill: Finaliza un proceso específico. Por ejemplo, kill 1234 termina el proceso con el ID 1234.

Información del sistema

Información del sistema Comandos útiles para obtener detalles sobre el uso del sistema de archivos, la memoria y la información general del sistema operativo en Linux.

df -h: Muestra el uso del espacio en disco de manera legible para el usuario (en GB/MB). Por ejemplo, df -h indica cuánto espacio total y libre hay en cada sistema de archivos.

free -h: Muestra el uso de la memoria RAM y swap en el sistema, con valores legibles para el usuario. Por ejemplo, free -h te permite ver cuánta memoria está en uso, libre o en caché.

uname -a: Muestra información del sistema operativo, incluyendo el nombre del kernel, la versión y la arquitectura. Por ejemplo, uname -a proporciona detalles completos sobre el sistema operativo en uso.

Redes

Comandos básicos para obtener información de la red y verificar la conectividad en Linux.

ifconfig: Muestra la configuración de las interfaces de red en el sistema. Por ejemplo, ifconfig proporciona detalles como la dirección IP, la dirección MAC y el estado de la conexión de la red.

ping: Envía paquetes de prueba a un destino para verificar la conectividad de la red. Por ejemplo, ping google.com envía paquetes a "google.com" y muestra si hay respuesta, indicando que la conexión a internet funciona.
